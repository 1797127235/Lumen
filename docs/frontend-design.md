# CodePilot 前端设计提案 · Phase 1

> 范围:对话 / 画像 / JD 诊断 三页端到端能用。
> 风格:温柔学长 · 手写感 · 偏人文。仅深色。
> 状态:**待你审阅**。批准后才开 `app/frontend/` 写代码。

---

## 0. 一句话调性

> **它不是一个对话框,是学长在台灯下递给你的一本笔记。**
> 深色的纸面,楷体的字,暖琥珀的墨水。它不催你,也不跟你比谁更聪明。

具体到执行层面这意味着：信息密度低、留白多、单一暖色重点、字形带书写感、文案是"我跟你讲"而不是"系统提示您"。

## 1. 字体 · Typography

> 反对清单:Inter / Fraunces / Newsreader / Lora / Crimson / Playfair / Cormorant / Syne / IBM Plex / Space Mono / Space Grotesk / DM Sans / DM Serif / Outfit / Plus Jakarta / Instrument — 这些都被 impeccable 列为"训练数据反射,会让所有项目长得一样"。我们绕开。

| 用途 | 字体 | 来源 / 许可 | 备注 |
|---|---|---|---|
| 中文(主,标题+正文) | **霞鹜文楷 LXGW WenKai** | github.com/lxgw/LxgwWenKai · OFL | 楷书风,清晰,书写感强;UI 用 Screen 版,正文用常规版 |
| 拉丁(主,标题+正文) | **Alegreya** | Google Fonts · OFL | 阿根廷设计师 Juan Pablo del Peral,人文衬线带书法笔意,屏幕可读 |
| 数字 / 等宽 / 代码 | **JetBrains Mono** | jetbrains.com/lp/mono · OFL | 来自 JetBrains 团队,字形干净,带字距与连字,不是 IBM Plex 的工业感 |
| 手写花体(极少用) | **Caveat** | Google Fonts · OFL | 仅用于"一句话便签"或欢迎页签名,绝不大段使用 |

**字号尺度**(1.25 ratio,4pt 取整,中文为主调过)：

```
xs:    12px   /* 时间戳、辅助说明 */
sm:    14px   /* 标签、按钮 */
base:  17px   /* 正文,比常规 16 大一格 — 楷体小字会糊 */
lg:    21px   /* 副标题 */
xl:    26px   /* 区块标题 */
2xl:   33px   /* 页面大标题 */
3xl:   41px   /* 唯一一处大数字(JD 评分) */
```

**行高**：

```
紧凑(UI/按钮):    1.4
正文(中文):       1.85   /* 楷体笔画密,需要更松 */
正文(英文):       1.65
显示标题:         1.25
```

**字重**：Alegreya 用 400 / 500 / 700;LXGW WenKai 只有一个字重(本身就是这个项目的特色),通过字号建立层次。

## 2. 调色板 · OKLCH

> 暖墨水版。中性灰统一向 60° (温暖琥珀方向)轻度倾斜 — chroma 0.010-0.015,人眼几乎察觉不到色相但整体氛围会暖一格。

```css
:root {
  /* 基础层 */
  --bg:                oklch(0.18  0.010  60);   /* 旧本子封皮的极深棕黑 */
  --surface:           oklch(0.22  0.012  60);   /* 一张纸 */
  --surface-elevated:  oklch(0.27  0.015  60);   /* 翻起来的一角 */

  /* 边线层 */
  --border-soft:       oklch(0.32  0.012  60);   /* 浅压痕 */
  --border:            oklch(0.40  0.014  60);   /* 钢笔细线 */

  /* 文字层 */
  --text:              oklch(0.94  0.015  60);   /* 暖白纸色 */
  --text-muted:        oklch(0.72  0.012  60);   /* 旁批小字 */
  --text-subtle:       oklch(0.55  0.010  60);   /* 标尺刻度 */

  /* 唯一重点色:琥珀墨水 */
  --ink:               oklch(0.78  0.140  75);   /* 主重点色 */
  --ink-deep:          oklch(0.58  0.150  70);   /* hover/pressed */
  --ink-soft:          oklch(0.88  0.060  80);   /* 高亮/标签底 */

  /* 状态色(克制) */
  --success:           oklch(0.74  0.090  150);  /* 暗鼠尾草绿 */
  --danger:            oklch(0.68  0.150  25);   /* 暖珊瑚,不是企业红 */
}
```

**用色规则**：

- 重点色 `--ink` 全屏出现频率不超过 5% 像素 — "用得越少越有力量"。
- 文字永远用单色,**禁用渐变文字**(impeccable absolute ban)。
- 卡片/列表项**禁用左/右 stripe 色边**(impeccable absolute ban) — 用全 1px `--border-soft` 边框、或纯 `--surface` 底色区分。
- danger 红只在真错误时出现,不用于"必填"标记(必填用文案"必填"二字而不是星号红色)。

## 3. 间距 / 圆角 / 边线

```css
/* 4pt 系统,语义命名 */
--space-2xs: 4px;
--space-xs:  8px;
--space-sm:  12px;
--space-md:  16px;
--space-lg:  24px;
--space-xl:  32px;
--space-2xl: 48px;
--space-3xl: 64px;
--space-4xl: 96px;

/* 圆角:略大,贴"软纸"感 */
--radius-sm: 4px;    /* 标签 */
--radius-md: 8px;    /* 输入框、按钮 */
--radius-lg: 14px;   /* 卡片 */
--radius-xl: 20px;   /* 主对话气泡 */

/* 边线宽度:全局只有 1px,不允许更粗(stripe ban) */
--border-width: 1px;
```

**布局宽度**：

- 全局最大宽度容器:`max-w-[820px]`(JD 页),`max-w-[720px]`(Profile),`max-w-[680px]`(Chat 主流) — 严格遵守 65-75ch 阅读上限。
- 顶部导航高度:52px,左右内边距 24px(桌面)/ 16px(移动)。

## 6. 文案口吻 · Microcopy

**原则**：动词在前+具体宾语,绝不出现"提交/确认/保存"。

| 场景 | 文字 |
|---|---|
| 上传简历按钮 | 拖一份过来 / 选个文件 |
| 发送对话 | 输入框右侧 `↩` 箭头(无文字) |
| JD 提交按钮 | 让学长看看 |
| 保存画像 | 记下来 |
| 取消修改 | 再想想 |
| 添加技能 | + 添加一项 |
| 删除某项 | 划掉这条 |

**空状态**：

| 页面 | 文案 |
|---|---|
| Chat 首次进入 | "我在等你说第一句." |
| Profile 无数据 | "先扔一份简历给我,我看看你现在在哪一格." |
| JD 还没诊断 | "把你看上的那段贴上来,我们对一对." |

**加载状态**：

- SSE chat → 不显示 spinner,只在最后一条气泡里 `学长正在写...┃` (细光标淡入)。
- 简历解析(3-8 秒)→ 文字 "我在读..." + 下方一条 1px 进度条从左推到右(无百分比)。
- JD 诊断 → 文字 "我在比对..." + 同上进度条。

**错误状态**：

| 错误 | 文案 |
|---|---|
| 网络断开 | "信号断了一下,你再发一次试试?" |
| LLM 失败 | "我刚才走神了,你再问我一遍." |
| 文件格式不对 | "这种文件我读不来.要 PDF / DOCX / TXT / MD." |
| 文件超 10MB | "文件有点大,精简到 10MB 以内?" |
| 鉴权失败(占位) | "让我先认识一下你." |

**禁用词**：
- 系统/用户/请/您/亲/操作成功/数据已保存/正在处理/请稍候
- 任何"作为一个 AI 模型..."开头的句子
- 全大写英文单词(除了缩写如 `JD` `PDF` `STAR`)

## 7. 动效

| 类型 | 实现 |
|---|---|
| 缓动 | `cubic-bezier(0.16, 1, 0.3, 1)` (ease-out-expo) |
| 时长 | 200-280ms |
| 允许动的属性 | 只 `transform` + `opacity` |
| 禁止 | bounce / elastic / spring,不动 height/padding/margin |
| 主要动效场景 | 1) 消息气泡进入(translateY 8 → 0,opacity 0 → 1)<br>2) 学长正在写 光标(opacity 0.4 ↔ 1, 1.4s 循环)<br>3) 页面切换(opacity 180ms,无滑动) |
| reduced-motion | `@media (prefers-reduced-motion: reduce)` 全部关掉 |
