## 一、功能模块总览

### 1.1 一级模块划分

基于"计算机专业学生的AI职业规划伙伴"产品定位，结合4个阶段10个核心场景，系统将功能划分为 **5个一级模块**：

| 一级模块 | 模块定位 | 核心目标 | 覆盖用户旅程阶段 |
|---------|---------|---------|----------------|
| **职业认知与诊断** | 用户起点模块，帮助建立方向认知 | 解决"我是谁、适合什么方向" | 入学适应期 |
| **学习路径与成长** | 能力建设核心模块 | 解决"学什么、怎么学、学到什么程度" | 技能积累期 |
| **求职作战室** | 求职冲刺核心模块 | 解决"简历、面试、进度管理" | 实习求职期 |
| **AI对话伙伴** | 贯穿全旅程的交互入口 | 7x24小时答疑、陪伴、情感支持 | 全阶段 |
| **个人数据中心** | 用户资产沉淀模块 | 记录画像、进度、历史、成就 | 全阶段 |

### 1.2 一级模块下的二级功能清单

#### 模块A：职业认知与诊断

| 二级功能 | 功能描述 | 版本规划 |
|---------|---------|---------|
| A-01 行业全景导览 | 对话式计算机行业方向介绍（后端/前端/算法/DevOps/安全/数据等），含工作内容、技能要求、发展前景、薪资区间 | MVP |
| A-02 智能方向测评 | 融合技术偏好、性格特质、能力基础的互动式测评，生成个人-方向匹配雷达图和Top3推荐 | MVP -> S-04增强 |
| A-03 大学四年规划生成 | 基于目标方向、当前基础、院校层级、可用时间生成学期级规划，含里程碑验收标准 | MVP |
| A-04 行业数据看板 | 实时岗位数量、薪资分布、技能要求趋势（按方向/城市/公司层级） | 增强版 S-03 |

#### 模块B：学习路径与成长

| 二级功能 | 功能描述 | 版本规划 |
|---------|---------|---------|
| B-01 技术栈智能推荐 | 解析目标岗位JD，提取核心技术栈并标注"必须/加分/了解"三级优先级 | MVP |
| B-02 个性化学习路径生成 | 渐进式学习路径，按周/月拆解，标注预计时长、验收标准、学习资源 | MVP |
| B-03 岗位能力差距分析 | 目标岗位能力模型拆解 + 多维度自评/测试，生成能力雷达图与补全建议 | MVP |
| B-04 项目实践指导 | 差异化项目推荐、PRD设计、技术方案建议、代码Review | 增强版 S-01 |
| B-05 学习进度追踪 | 用户标记完成进度，系统自动追踪并生成进度报告 | 增强版 S-06 |
| B-06 动态路径调优 | 根据学习进度、测评结果、市场变化自动调整后续路径 | 增强版 S-06 |

#### 模块C：求职作战室

| 二级功能 | 功能描述 | 版本规划 |
|---------|---------|---------|
| C-01 简历智能解析 | 上传简历（PDF/Word/图片），提取结构化信息 | MVP |
| C-02 简历智能优化 | JD匹配度评分、关键词缺失提示、STAR法则优化项目描述、多模板输出 | MVP |
| C-03 技术八股模拟面试 | 高频题库（OS/网络/数据库/中间件/Java等），追问式对话，即时评分反馈 | MVP |
| C-04 算法模拟面试 | 手写代码环境+实时判题+思路讲解评分+复杂度分析 | MVP |
| C-05 系统设计训练 | 经典场景题库+架构图绘制引导+扩展追问 | 增强版 |
| C-06 行为面试训练 | 基于宝洁八大问+亚马逊领导力准则的STAR回答训练 | 增强版 |
| C-07 综合模拟面试 | AI扮演面试官，30-60分钟完整流程，生成评估报告 | 增强版 |
| C-08 求职进度管理 | 可视化看板（投递->笔试->面试->OC全流程），节点提醒，漏斗分析 | 增强版 S-02 |
| C-09 Offer决策分析器 | 7维度量化评分+长期发展预测+情景模拟+结构化对比报告 | 增强版 S-05 |
| C-10 校招日历 | 各大厂校招时间线聚合（提前批/正式批/补录），订阅提醒 | 生态版 C-03 |

#### 模块D：AI对话伙伴

| 二级功能 | 功能描述 | 版本规划 |
|---------|---------|---------|
| D-01 多轮职业咨询 | 自然语言询问方向、发展路径、学习建议，结构化个性化回答 | MVP |
| D-02 实时技术答疑 | 学习过程中随时提问（如"为什么Redis单线程还这么快"） | MVP |
| D-03 求职心理疏导 | 焦虑时提供情绪支持、策略调整建议、同辈对比安抚 | MVP |
| D-04 主动推送提醒 | 关键时间节点主动提醒（如"实习申请季开始""某厂笔试倒计时3天"） | 增强版 |
| D-05 多模态交互 | 语音对话、图片上传（手写笔记/简历拍照） | 生态版 C-06 |

#### 模块E：个人数据中心

| 二级功能 | 功能描述 | 版本规划 |
|---------|---------|---------|
| E-01 用户画像管理 | 基本信息、目标方向、当前技能、偏好设置 | MVP |
| E-02 学习档案 | 已完成的学习节点、测试成绩、能力雷达图历史 | MVP |
| E-03 求职档案 | 投递记录、面试记录、简历版本历史 | MVP |
| E-04 成就与里程碑 | 解锁的成就徽章、达成的里程碑、成长时间线 | 增强版 |
| E-05 数据导出 | 职业规划报告、能力评估报告、求职进度报告导出PDF/Word | 生态版 C-04 |

### 1.4 功能优先级与版本规划

#### MVP版本（第1阶段：0-3个月）— "方向与基础"

**目标**: 验证核心价值，覆盖最痛场景

| 模块 | 功能 | 交付物 |
|------|------|--------|
| 职业认知与诊断 | A-01 行业全景导览、A-02 智能方向测评（简化版）、A-03 四年规划生成 | 可对话询问方向、完成测评、生成规划 |
| 学习路径与成长 | B-01 技术栈推荐、B-02 学习路径生成、B-03 能力差距分析（简化版） | 根据目标生成路径、基础差距诊断 |
| 求职作战室 | C-01 简历解析、C-02 简历优化、C-03 技术八股面试（高频50题）、C-04 算法模拟面试（简化版） | 简历上传优化、基础模拟面试 |
| AI对话伙伴 | D-01 多轮职业咨询、D-02 实时技术答疑、D-03 求职心理疏导 | 全场景对话能力 |
| 个人数据中心 | E-01 用户画像、E-02 学习档案（简化）、E-03 求职档案（简化） | 基础数据管理 |

**MVP成功指标**:
- 用户首次对话完成率 >= 70%
- 学习路径生成后7日回访率 >= 40%
- 简历优化功能使用率 >= 25%

#### 增强版本（第2阶段：4-6个月）— "闭环与深度"

**目标**: 完善核心场景闭环，提升用户粘性

| 新增功能 | 价值 |
|---------|------|
| A-02 智能方向测评（完整版） | 融合霍兰德+技术偏好+学习风格 |
| A-04 行业数据看板 | 数据驱动的方向选择 |
| B-04 项目实践指导 | 解决"没项目写"痛点 |
| B-05 学习进度追踪 + B-06 动态路径调优 | 静态路径升级为动态自适应 |
| C-05 系统设计训练 + C-06 行为面试训练 + C-07 综合模拟面试 | 面试全科目覆盖 |
| C-08 求职进度管理看板 | 可视化作战室 |
| C-09 Offer决策分析器 | 多维度量化决策 |
| D-04 主动推送提醒 | 从被动咨询到主动陪伴 |
| E-04 成就与里程碑 | 游戏化激励 |

**增强版成功指标**:
- 用户周活跃率 >= 35%
- 模拟面试人均使用 >= 3次/月
- 求职进度管理秋招季渗透率 >= 30%

#### 生态版本（第3阶段：7-12个月）— "社区与生态"

| 新增功能 | 价值 |
|---------|------|
| C-10 校招日历 | 时间节点把控 |
| D-05 多模态交互 | 语音、图片输入 |
| E-05 数据导出 | 报告存档与沟通 |
| 社区/同辈交流 | 组队刷题、面经分享 |
| 导师1v1对接 | 高端增值服务 |
| 多租户SaaS | 高校批量采购 |
