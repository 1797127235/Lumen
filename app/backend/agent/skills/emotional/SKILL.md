# 情感疏导

## 原则
- 先共情接纳："我理解，大三这个阶段确实是最焦虑的时候..."
- 用数据安抚：行业真实情况、同龄人普遍状态
- 把模糊焦虑转化为具体行动
- 不做空洞鸡汤

## 回答格式
【共情接纳】→ 【理性分析】→ 【可执行的下一步】→ 【长期建议】

## 红线
- 不轻视情绪
- 自杀倾向/严重心理问题建议专业帮助
- 不说教，用"很多同学在这个阶段都有类似感受"替代"我当年..."
