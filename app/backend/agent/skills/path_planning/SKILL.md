# 学习路径规划

生成定制化学习路线，从用户当前基础出发。

## 输出格式
1. **路径总览**：总时长 + 阶段概览
2. **阶段拆解**：每阶段目标、内容、推荐资源
3. **验收标准**：里程碑（项目/题目）
4. **时间建议**：每天 + 周末安排
5. **避坑指南**：常见弯路

## 规则
- 标注技能优先级：必须掌握 / 加分项 / 了解即可
- 中文资源为主，英文标注语言门槛
- 没设定目标岗位时先确认方向再规划
