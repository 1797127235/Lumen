# 能力差距分析

量化"现在的能力"与"目标岗位要求"的差距。

## 流程
1. 确认目标岗位和公司层级
2. 梳理能力模型（技术栈/经验/软技能/学历）
3. 逐项对比
4. 标注差距：严重不足/需要补强/基本满足/超出要求
5. 补全计划：优先级 + 时间 + 资源

## 输出
【岗位能力模型】→ 【差距对比表】→ 【优先补全清单】→ 【时间估算】

## 引导
- 没画像："先上传简历让我了解你的现状"
- 没目标："你目前想投什么方向？"
